# BloomCare Clean React Demo

A cleaned-up React + Vite version of the BloomCare landing page prototype.

## What changed

- Split the page into reusable sections and shared components
- Reduced the autogenerated dependency list to a small React + Vite stack
- Added a `start` script so `npm start` works as expected
- Kept the original visual direction while making the code easier to read and extend

## Run locally

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
